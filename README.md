# 项目简介

## 核心功能
- 基于 YOLOv8 的中草药检测系统
- 支持 4800 张数据集，涵盖 38 类中草药
- 提供 38 种中草药分类检测功能
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和 IOU 阈值
- 可选择显示标签和原图

## 适用场景
- 中药材识别与分类
- 药材质量检测
- 药材品种鉴别
- 药材流通中的快速识别
- 药材库存管理
- 药材教学与研究

## 技术特点
- 基于 YOLOv8 目标检测算法
- 使用 Python 3.9 开发环境
- 集成 OpenCV、PyQt5、Ultralytics、PyTorch 1.9 等技术栈
- 提供图形用户界面
- 支持多种输入源（图片、视频、摄像头）
- 已训练好模型，可直接使用
